## 🎨 画面モック（iPad ブラウザ想定 / MVP）

画面は家庭内利用を想定し、子の操作は最小限（1トグル）。親パネルはPINロックで保護します。以下はワイヤーフレームのテキストモックです。

---

### 子ホーム（待機）

```
┌──────────────────────────── iPad Safari ────────────────────────────┐
│  残り時間                                                             │
│                                                                      │
│                         1 時間 30 分                                  │
│                                                                      │
│                                                                      │
│                         [ ゲームをはじめる ]                          │
│                                                                      │
│  状態: 待機中                                                         │
│                                                                      │
└──────────────────────────────────────────────────────────────────────┘
```

ポイント
- ボタンは画面幅の約60〜80%で大きく、誤タップ防止の余白を確保
- 残り0分ならボタンは無効化し、理由（残り時間がありません）を表示

---

### 子ホーム（実行中）

```
┌──────────────────────────── iPad Safari ────────────────────────────┐
│  残り時間                                                             │
│                                                                      │
│                         1 時間 12 分                                  │
│                                                                      │
│                    経過: 00:18:24  （バックグラウンドOK）              │
│                                                                      │
│                         [ 長押しで終了する ]                          │
│                                                                      │
│  状態: 実行中（開始: 19:02）                                          │
│                                                                      │
└──────────────────────────────────────────────────────────────────────┘
```

ポイント
- 終了は長押し or 確認ダイアログで誤操作防止
- 経過は目安表示。正計算は開始時刻→終了時刻の再計算で行う

---

### 親パネル（PIN ロック）

PIN 入力
```
┌──────────────────────────── iPad Safari ────────────────────────────┐
│  親モード                                                             │
│                                                                      │
│                     PIN を入力してください［ ＿ ＿ ＿ ＿ ］              │
│                                                                      │
│                              [ 開く ]                                 │
│                                                                      │
└──────────────────────────────────────────────────────────────────────┘
```

解錠後
```
┌──────────────────────────── iPad Safari ────────────────────────────┐
│  親モード                                                             │
│                                                                      │
│  残り時間: 1 時間 12 分                                              │
│                                                                      │
│  付与（プリセット）:   [ +5分 ]   [ +10分 ]   [ +15分 ]               │
│                                                                      │
│  リセット:            [ 0分にする ]  （確認ダイアログあり）            │
│                                                                      │
│  バックアップ:        [ エクスポート ]  [ インポート ]                 │
│                                                                      │
└──────────────────────────────────────────────────────────────────────┘
```

補足
- 付与/リセットは即時反映。操作時にトーストで通知（例: +10分 付与しました）
- バックアップはJSONファイル。インポート時は上書き確認を表示

---

### ナビゲーション（例）

```
[子ホーム] ── PINボタン → [親PIN入力] ── 成功 → [親パネル]
   │                                   └─ 失敗 → エラー表示/クールダウン
   └─ 実行中: 長押し終了 → 確認 → 精算 → 残り時間更新
```

---

このモックをもとに、最小PWAのUI実装（HTML/CSS/JS）に着手できます。必要ならカラーパレットやフォントサイズのガイドも追加します。
